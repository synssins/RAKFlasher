Serial Monitor Guide

  • HEX Mode: Shows binary data as hex dump — ideal for Meshtastic protobuf (PROTO) mode
  • TEXT Mode: Shows printable ASCII text — ideal for text-based serial protocols
  • Frame Detection: Meshtastic protobuf frames start with 94 C3 header bytes
  • Send Hex: Check "Send as raw hex bytes" to send binary data (e.g. 94 C3 00 02 18 01)
  • Passthrough Mode: Enable bidirectional serial bridge for CLI access
  • Download: Save the current buffer as a text file
