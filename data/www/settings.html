What's New

v1.2.4

  • TCP bridge no longer hangs after Meshtastic CLI commands (60s timeout + flush)
  • Fixed Meshtastic command console text input (channels, help, reboot N)
  • WiFi STA credentials now persist across OTA firmware updates

v1.2.3

  • SWD turnaround speed fix — replaced slow pinMode() with fast GPIO register writes

v1.2.2

  • “Flash Last Backup” button — restore firmware directly from device, no re-upload
  • Fixed stale “Operation completed” alert showing when starting new operations
  • Fixed Meshtastic command console duplicate polling / stuck busy state

v1.2.1

  • SWD backup speed dramatically improved (NOP-based timing + higher task priority)
  • Fixed download filename for SWD backups (was “download”, now proper .bin name)
  • Fixed repeated success alerts after operations complete
  • Real-time KB/s speed displayed during flash dump

v1.2.0

  • Auto-incrementing build numbers for easy version tracking
  • Changelog on settings page
  • Build number visible on Dashboard and Settings

v1.1.0

  • SWD firmware backup & restore (full flash dump via SWD)
  • Meshtastic command console with AT command support
  • TCP serial bridge for Meshtastic client apps (port 4403)
  • WiFi STA (client) mode — join existing networks
  • OTA firmware updates (single .bin includes web UI)
  • Configurable serial baud rate
  • Complete SWD engine rewrite based on proven reference

v1.0.0

  • Initial release
  • Serial DFU flashing, SWD recovery, WiFi AP
  • Web UI with 6 pages, serial monitor, GPIO control
  • Configuration backup/restore framework
