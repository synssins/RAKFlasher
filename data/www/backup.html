Backup & Restore Guide

Full Settings Backup:

  • Captures all device configuration, module settings, and user information
  • Stored as JSON format for easy viewing and editing
  • Recommended before firmware updates

Channel & Keys Backup:

  • Stores channel configurations, pre-shared keys (PSKs), and encryption keys
  • Can be encrypted with a password for security
  • Separate from settings to allow sharing without exposing keys
Important: Keep your backups secure, especially channel backups which contain encryption keys!
